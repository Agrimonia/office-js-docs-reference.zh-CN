# <a name="description-element"></a>Description 元素

指定 Office 外接程序的说明为不超过 250 个字符的字符串。

**外接程序类型：** 内容、任务窗格、邮件

## <a name="syntax"></a>语法

```XML
<Description DefaultValue="string">
```
