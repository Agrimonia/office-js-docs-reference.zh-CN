# <a name="citationtext-element"></a>CitationText 元素

指定该词典的引文样本文字。

**外接程序类型：** 内容、任务窗格、邮件

## <a name="syntax"></a>语法

```XML
<CitationText DefaultValue="string" />
```

## <a name="contained-in"></a>包含在

"词典"
