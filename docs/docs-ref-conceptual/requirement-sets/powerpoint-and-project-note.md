# <a name="powerpoint-and-project-in-the-javascript-api"></a>PowerPoint 和 JavaScript API 中的项目

PowerPoint 和 Project 支持使用 JavaScript API 所做的加载项。 但是，他们当前不具有特定主机的 Api。 您与这些主机通过共享 API 进行交互。
